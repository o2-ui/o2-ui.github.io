# Installation

## Requirements

`@o2-ui/carmore-icon` expects the following environment:

- React 18 or newer
- React DOM 18 or newer

## Install

```bash
# npm
npm install @o2-ui/carmore-icon

# yarn
yarn add @o2-ui/carmore-icon

# pnpm
pnpm add @o2-ui/carmore-icon
```
