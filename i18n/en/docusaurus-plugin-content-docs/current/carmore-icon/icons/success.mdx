## Size

Adjust the icon size with the `width` and `height` props.

- Pass numbers directly, or strings that include units such as 'px', 'rem', or '%'.
  - (The original aspect ratio is fixed and cannot be changed.)
- The default `width` is '1.5rem'.
- The default `height` is '1.5rem'.

<SizePropsCompareList icon={'success'} sizes={['1.5rem', 30, '48px', '5rem']} />

```tsx
<CarmoreIcon icon={'success'} ... /> // Uses the default ('1.5rem')
<CarmoreIcon icon={'success'} width={30} height={30} ... />
<CarmoreIcon icon={'success'} width={'48px'} height={'48px'} ... />
<CarmoreIcon icon={'success'} width={'5rem'} height={'5rem'} ... />
```

## Accessibility (a11y)

Set attributes such as `role` and `aria-label` for better accessibility.

- The default `role` is 'img'.
- The default `aria-label` is '标车'.
- Default accessibility props are provided, and you can customize them as needed.
- _(Hover over the icon.)_

<A11yPropsCompareList icon={'success'} ariaLabels={[undefined, 'Approved', 'Check mark']} />

```tsx
<CarmoreIcon icon={'success'} ... /> // Uses the default ('标车')
<CarmoreIcon icon={'success'} aria-label={'Approved'} ... />
<CarmoreIcon icon={'success'} aria-label={'Check mark'} ... />
```
