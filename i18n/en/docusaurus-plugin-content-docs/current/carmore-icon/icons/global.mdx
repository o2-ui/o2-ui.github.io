## Size

Adjust the icon size with the `width` and `height` props.

- Pass numbers directly, or strings that include units such as 'px', 'rem', or '%'.
  - (The original aspect ratio is fixed and cannot be changed.)
- The default `width` is '1.5rem'.
- The default `height` is '1.5rem'.

<SizePropsCompareList icon={'global'} sizes={['1.5rem', 30, '48px', '5rem']} />

```tsx
<CarmoreIcon icon={'global'} ... /> // Uses the default ('1.5rem')
<CarmoreIcon icon={'global'} width={30} height={30} ... />
<CarmoreIcon icon={'global'} width={'48px'} height={'48px'} ... />
<CarmoreIcon icon={'global'} width={'5rem'} height={'5rem'} ... />
```

## Color

Set `className` with a text color to change the icon color.

<ColorPropsCompareList icon={'global'} colors={['red', 'orange', 'green', 'blue', 'purple']} />

```css

  color: red;
}
```

## Accessibility (a11y)

Set attributes such as `role` and `aria-label` for better accessibility.

- The default `role` is 'img'.
- The default `aria-label` is '글로벌'.
- Default accessibility props are provided, and you can customize them as needed.
- _(Hover over the icon.)_

<A11yPropsCompareList icon={'global'} ariaLabels={[undefined, 'Global', 'Internet']} />

```tsx
<CarmoreIcon icon={'global'} ... /> // Uses the default ('글로벌')
<CarmoreIcon icon={'global'} aria-label={'Global'} ... />
<CarmoreIcon icon={'global'} aria-label={'Internet'} ... />
```
