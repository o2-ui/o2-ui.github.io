## Control size

You can control the icon size with the `width` and `height` props.

- Pass numbers directly, or strings with units such as 'px', 'rem', or '%'.
  - (The icon keeps its intrinsic aspect ratio and cannot be stretched.)
- The default `width` is '1.5rem'.
- The default `height` is '1.5rem'.

<SizePropsCompareList icon={'global'} sizes={['1.5rem', 30, '48px', '5rem']} />

```tsx
<CarmoreIcon icon={'global'} ... /> // Uses the default ('1.5rem')
<CarmoreIcon icon={'global'} width={30} height={30} ... />
<CarmoreIcon icon={'global'} width={'48px'} height={'48px'} ... />
<CarmoreIcon icon={'global'} width={'5rem'} height={'5rem'} ... />
```

## Change color

Set the text color via the `className` prop to recolor the icon.

<ColorPropsCompareList icon={'global'} colors={['red', 'orange', 'green', 'blue', 'purple']} />

```css
.text-red {
  color: red;
}

.text-orange {
  color: orange;
}

.text-green {
  color: green;
}

.text-blue {
  color: blue;
}

.text-purple {
  color: purple;
}
```

```tsx
<CarmoreIcon icon={'global'} className={'text-red'} ... />
<CarmoreIcon icon={'global'} className={'text-orange'} ... />
<CarmoreIcon icon={'global'} className={'text-green'}  ... />
<CarmoreIcon icon={'global'} className={'text-blue'}  ... />
<CarmoreIcon icon={'global'} className={'text-purple'}  ... />
```

## Accessibility (a11y)

We recommend setting accessibility attributes such as `role` and `aria-label`.

- The default `role` is 'img'.
- The default `aria-label` is 'Worldwide'.
- Accessibility defaults are provided, and you can customize them as needed.
- _(Hover the icon to see the tooltip.)_

<A11yPropsCompareList icon={'global'} ariaLabels={[undefined, 'Global', 'Internet']} />

```tsx
<CarmoreIcon icon={'global'} ... /> // Uses the default ('Worldwide')
<CarmoreIcon icon={'global'} aria-label={'Global'} ... />
<CarmoreIcon icon={'global'} aria-label={'Internet'} ... />
```
