## Control size

You can control the icon size with the `width` and `height` props.

- Pass numbers directly, or strings with units such as 'px', 'rem', or '%'.
  - (The icon keeps its intrinsic aspect ratio and cannot be stretched.)
- The default `width` is '1.5rem'.
- The default `height` is '1.5rem'.

<SizePropsCompareList icon={'hotel-room'} sizes={['1.5rem', 30, '48px', '5rem']} />

```tsx
<CarmoreIcon icon={'hotel-room'} ... /> // Uses the default ('숙실')
<CarmoreIcon icon={'hotel-room'} width={30} height={30} ... />
<CarmoreIcon icon={'hotel-room'} width={'48px'} height={'48px'} ... />
<CarmoreIcon icon={'hotel-room'} width={'5rem'} height={'5rem'} ... />
```

## Accessibility (a11y)

We recommend setting accessibility attributes such as `role` and `aria-label`.

- The default `role` is 'img'.
- The default `aria-label` is '숙실'.
- Accessibility defaults are provided, and you can customize them as needed.
- _(Hover the icon to see the tooltip.)_

<A11yPropsCompareList icon={'hotel-room'} ariaLabels={[undefined, 'Bedroom', 'Hotel']} />

```tsx
<CarmoreIcon icon={'hotel-room'} ... /> // Uses the default ('숙실')
<CarmoreIcon icon={'hotel-room'} aria-label={'Bedroom'} ... />
<CarmoreIcon icon={'hotel-room'} aria-label={'Hotel'} ... />
```
