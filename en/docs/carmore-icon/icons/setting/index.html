Setting


Usage​

import { CarmoreIcon } from '@o2-ui/carmore-icon';

// ...

  <CarmoreIcon icon={'setting'} ... />

// ...


Control size​

You can control the icon size with the width and height props.

  • Pass numbers directly, or strings with units such as 'px', 'rem', or '%'.
    • (The icon keeps its intrinsic aspect ratio and cannot be stretched.)
  • The default width is '1.5rem'.
  • The default height is '1.5rem'.
<CarmoreIcon icon={'setting'} ... /> // Uses the default ('설정하기')
<CarmoreIcon icon={'setting'} width={30} height={30} ... />
<CarmoreIcon icon={'setting'} width={'48px'} height={'48px'} ... />
<CarmoreIcon icon={'setting'} width={'5rem'} height={'5rem'} ... />


Change color​

Set the text color via the className prop to recolor the icon.

.text-red {
  color: red;
}

.text-orange {
  color: orange;
}

.text-green {
  color: green;
}

.text-blue {
  color: blue;
}

.text-purple {
  color: purple;
}
<CarmoreIcon icon={'setting'} className={'text-red'} ... />
<CarmoreIcon icon={'setting'} className={'text-orange'} ... />
<CarmoreIcon icon={'setting'} className={'text-green'}  ... />
<CarmoreIcon icon={'setting'} className={'text-blue'}  ... />
<CarmoreIcon icon={'setting'} className={'text-purple'}  ... />


Accessibility (a11y)​

We recommend configuring attributes such as role and aria-label.

  • The default role is 'img'.
  • The default aria-label is '설정하기'.
  • Base accessibility defaults are provided, and you can override them when needed.
  • (Hover the icon to check the tooltip.)
<CarmoreIcon icon={'setting'} ... /> // Uses the default ('설정하기')
<CarmoreIcon icon={'setting'} aria-label={'Edit'} ... />
<CarmoreIcon icon={'setting'} aria-label={'Gear'} ... />
