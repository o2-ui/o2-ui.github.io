## 크기 설정

아이콘의 크기는 `width` 와 `height` 속성으로 조절할 수 있습니다.

- 숫자를 직접 지정할 수 있으며, px, rem, % 같은 단위가 포함된 문자열도 허용됩니다.
  - (단, 아이콘의 원래 비율은 고정되어 있어 변경할 수 없습니다.)
- `width` 의 기본값은 '1.5rem' 입니다.
- `height` 의 기본값은 '1.5rem' 입니다.

<BorderBox>
  <Flex direction={'row'} gap={'2rem'} alignItems={'center'} justifyContent={'center'}>
    <CarmoreIcon className={'textPrimary'} icon={'arrow-down'} />
    <CarmoreIcon icon={'arrow-down'} width={30} height={30} />
    <CarmoreIcon icon={'arrow-down'} width={'3rem'} height={'3rem'} />
    <CarmoreIcon icon={'arrow-down'} width={'5rem'} height={'5rem'} />
  </Flex>
</BorderBox>

```tsx
<CarmoreIcon icon={'arrow-down'} ... /> // 기본값('1.5rem') 적용
<CarmoreIcon icon={'arrow-down'} width={30} height={30} ... />
<CarmoreIcon icon={'arrow-down'} width={'48px'} height={'48px'} ... />
<CarmoreIcon icon={'arrow-down'} width={'5rem'} height={'5rem'} ... />
```

## 색상 변경

`className`을 통해 텍스트 색상(color)을 지정하면 아이콘의 색상이 변경됩니다.

- 기본 색상은 프로젝트의 전역 CSS 설정에 따라 결정됩니다.
- 일부 아이콘은 색상 변경이 제한될 수 있으므로, 해당 아이콘의 상세 페이지를 참고해야 합니다.

<BorderBox>
  <Flex direction={'row'} gap={'2rem'} alignItems={'center'} justifyContent={'center'}>
    <CarmoreIcon className={'textRed'} icon={'arrow-down'} width={60} height={60} />
    <CarmoreIcon className={'textOrange'} icon={'arrow-down'} width={60} height={60} />
    <CarmoreIcon className={'textGreen'} icon={'arrow-down'} width={60} height={60} />
    <CarmoreIcon className={'textBlue'} icon={'arrow-down'} width={60} height={60} />
    <CarmoreIcon className={'textPurple'} icon={'arrow-down'} width={60} height={60} />
  </Flex>
</BorderBox>

```css
.text-red {
  color: red;
}

.text-orange {
  color: orange;
}

.text-green {
  color: green;
}

.text-blue {
  color: blue;
}

.text-purple {
  color: purple;
}
```

```tsx
<CarmoreIcon icon={'arrow-down'} className={'text-red'} ... />
<CarmoreIcon icon={'arrow-down'} className={'text-orange'} ... />
<CarmoreIcon icon={'arrow-down'} className={'text-green'}  ... />
<CarmoreIcon icon={'arrow-down'} className={'text-blue'}  ... />
<CarmoreIcon icon={'arrow-down'} className={'text-purple'}  ... />
```

## 접근성 (a11y)

`role`, `aria-label` 등의 속성을 설정하는 것을 권장합니다.

- `role` 의 기본값은 'img' 입니다.
- `aria-label` 의 기본값은 '아래로' 입니다.
- 기본 접근성 속성이 적용되어 있지만, 필요에 따라 커스터마이징할 수 있습니다.
- _(아이콘에 마우스를 올려보세요.)_

<BorderBox>
  <Flex direction={'row'} gap={'4rem'} alignItems={'center'} justifyContent={'center'}>
    <CarmoreIcon className={'textPrimary'} icon={'arrow-down'} width={60} height={60} />
    <CarmoreIcon icon={'arrow-down'} aria-label={'화살표'} width={60} height={60} />
    <CarmoreIcon icon={'arrow-down'} aria-label={'아래쪽 화살표'} width={60} height={60} />
  </Flex>
</BorderBox>

```tsx
<CarmoreIcon icon={'arrow-down'} ... /> // 기본값('아래로') 적용
<CarmoreIcon icon={'arrow-down'} aria-label={'화살표'} ... />
<CarmoreIcon icon={'arrow-down'} aria-label={'아래쪽 화살표'} ... />
```
